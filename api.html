API Key Information

What is my API Key?

Pokefarm Q allows you to generate an API key to access read only information about your account via their API.

How do I generate it?

You can find your API key here. You'll have to generate a new one if you don't already have one.

Anybody with access to this key can access read-only information about your account. Generate a new one if you're worried about it.

What do you do with my API Key?

All this site does is request your boost information from Pokefarm Q directly.
Things like whether you have Hypermode, your Albino Radar level, your current chain, and more.

This site does not store your API Key in any way; it's only used to get information when you press the calculate button.
